# Maison Altus — Site vitrine

Site vitrine pour Maison Altus, studio de design d'intérieur spécialisé en style japandi et wabi-sabi.

## Lancer le site en local

Ouvre simplement `index.html` dans ton navigateur, ou utilise un serveur local :

```bash
# Python
python3 -m http.server 8080

# Node (npx)
npx serve .
```

Puis ouvre [http://localhost:8080](http://localhost:8080).

## Déploiement

Compatible GitHub Pages, Netlify et Vercel. Aucun build requis.
